Instrucciones de Uso
🔧 Desarrollo Local:
  1. La página funciona con credenciales en env-config.js
  2. Haz clic en "Solicitar Permisos" para habilitar notificaciones
  3. Usa "Probar Notificación Local" primero
  4. HTTPS requerido para OneSignal (producción)
🚀 Producción:
  1. Ejecuta: cd alertas && node config-server.js
  2. Sube service worker a la raíz del dominio
  3. Verifica HTTPS habilitado
  4. Prueba notificaciones push
Tip: En desarrollo local, las notificaciones OneSignal pueden no funcionar completamente. Usa "Probar Notificación Local" para verificar que los permisos funcionan.
